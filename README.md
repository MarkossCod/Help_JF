Deslizamentos em Juiz de Fora – Plataforma Web Educacional
📄 Visão Geral

Este repositório contém o desenvolvimento de uma plataforma web educacional dedicada ao estudo dos deslizamentos no município de Juiz de Fora.

A aplicação tem como finalidade fornecer conteúdo técnico estruturado sobre riscos geotécnicos urbanos, com ênfase na contextualização local. O projeto foi concebido para atender demandas acadêmicas, servindo como material de apoio para estudantes, docentes e pesquisadores em formação.

🎯 Objetivo

O principal objetivo da plataforma é consolidar informações relevantes sobre deslizamentos de encostas, abordando:

Conceitos fundamentais relacionados a movimentos de massa;

Fatores condicionantes (geológicos, geomorfológicos, climáticos e antrópicos);

Impactos socioambientais associados;

Contextualização do cenário urbano de Juiz de Fora;

Importância da prevenção e da educação para mitigação de riscos.

🌍 Contexto Geotécnico

O município apresenta áreas com diferentes níveis de suscetibilidade a movimentos de massa, influenciadas por:

Ocupação urbana em áreas de encosta;

Regimes pluviométricos intensos e concentrados;

Alterações antrópicas no uso e cobertura do solo;

Limitações na infraestrutura de drenagem.

A plataforma organiza essas informações de forma técnica, objetiva e acessível ao público estudantil.
